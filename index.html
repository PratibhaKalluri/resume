Work

ETL process

Extraction of data from various sources like SQL server, SITS, Excel sheets to perform manipulations and transforming on the data in ALTERYX. Using tools such joins, formula to create measures, Macros to automate the various workflows, summarize tools to drill the data to gain insights, cross-tab/transpose tools to structure the data into the required form. After all the transformation, loading the structured data into Power BI to create dashboards and reports which are useful to take strategic decisions.

Supporting the admission teams by creating data models to monitor live data resulting in the improvement of business processes and take data-driven decisions. Creating calculated measures to forecart and set targets for the upcoming admission cycles.

POWER BI

Building dynamic, user-friendly dashboards and report in Power BI for technical as well as business users which helps the leadership team take actionable data driven decisions. Reports focusing on Agent Performances(similar to Sales reports), Enrolment monitoring, Application Processing Time Report and many more help improve admission porcess, monitor the performances of the agents, retention rates, Application to enrolment conversions rates.

Maintaining the suite of reports in a Power BI by regularly updating the data and providing updated dashboards for admissions, planning and leadership teams. Using DAX to create visual based measures,bookmarks to create instances, creating relationships for work with cross functional team data.
